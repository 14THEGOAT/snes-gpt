## 📋 What is snes-gpt?

snes-gpt is a small AI program, built in assembly language to run on the Super Nintendo (SNES) console. It brings a simple, lightweight version of GPT-style text generation to older gaming hardware. This project shows how AI can work on classic machines without modern processors.

You don’t need to know programming to try it out. This guide will help you download and run snes-gpt step-by-step.

## 🧰 Troubleshooting

If you run into problems, check these common issues:

- **ROM won’t load on emulator:** Ensure you downloaded the correct file type (`.smc` or `.sfc`). Update your emulator to the latest version.  
- **Game crashes or freezes:** Try restarting the emulator or your SNES console. Confirm your ROM file is not corrupted by re-downloading.  
- **No response to input:** Make sure your controller or keyboard is connected and recognized by the system.  
- **Screen displays garbage or noise:** Your emulator or console might not support the ROM properly. Try a different emulator or hardware.
